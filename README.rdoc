== Installation

  Clone repository:

  git clone git@github.com:sushengloong/dotvim.git ~/.vim

  Create symlinks:

  ln -s ~/.vim/vimrc ~/.vimrc

  Switch to the `~/.vim` directory, and fetch submodules:

  cd ~/.vim
  git submodule init
  git submodule update
