== Installation

Clone repository:
  git clone https://github.com/sushengloong/dotvim.git ~/.vim

Create symlinks:
  ln -s ~/.vim/vimrc ~/.vimrc

Setup Vundle:
  git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

Install bundle:
  vim +BundleInstall +qall
